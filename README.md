# Genealogy

## Python 3.10.8 with Flask and MVC design pattern

## Using virtual environment

```
python3 -m venv env
. env/bin/activate
deactivate
```

## Install requirements

```
pip install -r requirements.txt
```

## Start database

```
docker-compose -f docker/mysql.yml up -d
```

## Start server

```
python server.py
```
